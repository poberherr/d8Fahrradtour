# Fahrradtour — 20-Jähriges Jubiläum 🚴

Eintages-Radtour für eine Gruppe alter Schulfreunde. Volle Rursee-Runde im Nationalpark Eifel, dann Grillen mit Familien & Kids.

**Gruppe:** Männer, Anfang/Mitte 40, gemütliches Tempo
**Start:** Erkelenz, 09:00 Uhr
**Ziel:** Imtal 86, 41189 Mönchengladbach (Wanlo)
**Ankunft:** ~16:00 — Grillen bei Tageslicht mit Familien
**Logistik:** Auto + Fahrradanhänger (kein Zug-Stress)
**Saison:** Schwimmen im Rursee nur Mai–September

## Notfallplan: Falls die Zeit knapp wird

1. **Rursee-Fähre nutzen** — Von Rurberg oder Woffelsbach zurück nach Schwammenauel, spart 10–15 km
2. **Schwimmstopp kürzen** — 20 Min statt 45 Min
3. **Eis-/Kaffeestopp in Einruhr weglassen** — Direkt durchradeln
4. **Mittagessen kürzer halten** — 45 Min statt 1 Std
